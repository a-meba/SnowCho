## 프로젝트 소개
우리의 플랫폼은 가맹점들이 원활한 운영을 할 수 있도록 다양한 기능과 지원을 제공합니다. <br>
이를 통해 가맹점들은 효율적으로 주문을 관리하고 고객들에게 신속하고 정확한 서비스를 제공할 수 있습니다.


## 주요 기능과 특징
- 가맹점 관리: 각 가맹점의 정보, 위치, 호점 등을 체계적으로 관리할 수 있습니다.
- 주문 처리: 가맹점에서 들어온 주문을 신속하게 처리하고, 주문 상태를 실시간으로 확인할 수 있습니다.
- 재고 관리: 본사에서는 모든 가맹점의 상품 재고를 통합적으로 확인하고, 주문에 따른 재고 조절이 자동으로 이루어집니다.
- 매출 관리: 가맹점에서 발생한 매출 내역을 기록하고, 통계를 통해 매출 동향을 파악할 수 있습니다.

## 사용되는 기술 및 언어
- 언어: Java, JSP
- 프레임워크: Spring
- 데이터베이스: Oracle
- 프론트엔드: HTML, CSS, JavaScript

## 프로젝트 목표
이 프로젝트의 목표는 프렌차이즈의 가맹점과 본사 간의 원활한 의사 소통과 업무 효율성을 높이는 데에 있습니다. <br>
각 기능은 사용자 편의성과 안정성을 고려하여 개발되었으며, 시스템을 통해 프렌차이즈의 운영이 보다 효율적으로 이루어질 것입니다.
