## 📌 프로젝트 소개
전통적인 순두부 요리에 현대적인 브랜딩을 더한 **몽글두부**는 본사와 가맹점 사이의 복잡한 주문/물류 과정을 한눈에 파악할 수 있도록 돕습니다. 전화나 문자 대신 전용 시스템을 통해 식자재를 발주하고, 실시간 매출 통계를 확인하여 매장 운영의 효율성을 극대화합니다.

## ✨ 주요 기능 및 특징

### 🏠 스마트 가맹점 관리
* **전국 가맹점 데이터베이스**: 위치, 연락처, 매장 규모 등 상세 정보 관리
* **매장 상태 제어**: 본사 관리자의 가맹 계약 승인 및 영업 상태 실시간 변경

### 🥛 신선 물류 발주 시스템
* **간편 발주**: 매일 아침 생산되는 순두부와 전용 양념장의 디지털 주문
* **배송 트래킹**: 결제 완료부터 매장 도착까지의 물류 현황 모니터링

### 📊 재고 및 매출 대시보드
* **자동 재고 계산**: 매장 내 식자재 소진에 따른 재고 수량 업데이트
* **데이터 시각화**: 일별/월별 매출 데이터를 그래프로 변환하여 성과 분석 지원

## 🛠 사용 기술 및 환경

* **Backend**: Java, JSP, Spring Framework
* **Database**: Oracle Cloud Database (SQL Developer)
* **Frontend**: HTML5, CSS3, JavaScript, Recoil (State Management)
* **Server**: Apache Tomcat 9.0
